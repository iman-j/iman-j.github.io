Stand-alone projects

Bistodo - Data Analytics Dashboard

GitHub Page: Bistodo

Live Page: Coming Soon

A data analytics dashboard built using Django and Plotly Dash. I started this project before the new and what seems to be a fully complete framework django-plotly-dash came into play. This was a challenging project as Django and Plotly Dash are independent applications that run on their own server. My approach to the solution was to allow the Dash app run within the Django rendered page. This allowed Django to handle all routing and data models while also being able to pass context to the Dash app via the url.

Notebooks

Popular Clustering Algorithms in R and Python Running One Notebook

GitHub Page: Notebook

As you may know, clustering is an approach to unsupervised machine learning. In this notebook, I demonstrate the ease of use of R and Python when creating the three most popular clustering algorithms - K-Means, Hierarchical Clustering and Density-Based Spatial Clustering of Applications with Noise (DBSAN). To be able to compare R and Python clustering, this notebook demonstrates how to run R and Python in one Jypyter notebook. This is currently work in progress.


Archibald Prize Analysis

GitHub Page: Notebook

The Archibald Prize is an Australian award that is annually awarded to the best portrait.
